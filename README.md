# Apriori-Algorithm
Fast implementation of the Apriori Algorithm in Java.

Input:

* A text file of transactions with one transaction (of multiple items space seperated) on a single line.

* Minimum support as a value.(Not percentage)

* Minimum size of the item set.

* Name of output file.

Run the algorithm using the following command from the command line:

```
java Apriori input.txt minsupport sizeOfItemSet output.txt  
```
